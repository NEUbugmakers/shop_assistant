## shop_assistant


### 简介

这是一个C语言课程设计的项目。
预计项目采用**Qt框架**，**图形界面**，以及一些中小型超市日常所用到的一些功能其中包含：
- 账户管理
- 超市会员管理
- 超市商品信息及其库存管理
- 超市人事管理
- 可操作的促销活动设置
- 数据备份及恢复
